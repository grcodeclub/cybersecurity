Αγγλικά Ελληνικά Περιγραφή
CookieCookieΜικρό αρχείο δεδομένων που αποθηκεύεται στον υπολογιστή του χρήστη από έναν ιστότοπο για να θυμάται τις ρυθμίσεις και τις προτιμήσεις του χρήστη.
DoxingΦακέλωμα/ΞεσκέπασμαΗ δημοσιοποίηση προσωπικών πληροφοριών ενός ατόμου, όπως διευθύνσεις, τηλέφωνα και άλλες ευαίσθητες πληροφορίες, συνήθως χωρίς τη συγκατάθεσή του.
Keystroke LoggingΚαταγραφή ΠληκτρολογήσεωνΗ διαδικασία καταγραφής των πληκτρολογήσεων ενός χρήστη με σκοπό την κλοπή ευαίσθητων πληροφοριών, όπως κωδικοί πρόσβασης.
Logic BombΛογική ΒόμβαΜια κακόβουλη κώδικα που ενεργοποιείται όταν πληρούνται συγκεκριμένες συνθήκες ή προϋποθέσεις, προκαλώντας ζημιά ή αποτυχία στο σύστημα.
PayloadΦορτίοΗ κακόβουλη ενέργεια ή το περιεχόμενο ενός κακόβουλου λογισμικού που εκτελείται όταν το λογισμικό έχει διεισδύσει στο σύστημα.
Remote AccessΑπομακρυσμένη ΠρόσβασηΗ δυνατότητα πρόσβασης και διαχείρισης ενός υπολογιστή ή συστήματος από απομακρυσμένη τοποθεσία μέσω του δικτύου ή του διαδικτύου.
Antivirus SoftwareΛογισμικό ΑντιϊκούΕφαρμογές που ανιχνεύουν, αποτρέπουν και απομακρύνουν κακόβουλο λογισμικό από υπολογιστές και δίκτυα.
FirewallΤείχος ΠροστασίαςΈνα σύστημα ασφάλειας που ελέγχει την είσοδο και έξοδο της κυκλοφορίας δικτύου, προκειμένου να εμποδίσει μη εξουσιοδοτημένη πρόσβαση.
MalwareΚακόβουλο ΛογισμικόΟποιοδήποτε λογισμικό σχεδιάστηκε για να προκαλέσει βλάβη ή να αποκτήσει μη εξουσιοδοτημένη πρόσβαση σε συστήματα υπολογιστών, όπως ιοί, worms και trojans.
Penetration TestingΔοκιμή ΔιείσδυσηςΜια διαδικασία αξιολόγησης της ασφάλειας ενός συστήματος ή δικτύου, κατά την οποία εκτελούνται εικονικές επιθέσεις για να εντοπιστούν ευπάθειες.
Intrusion Detection SystemΣύστημα Ανίχνευσης ΕισβολώνΈνα σύστημα που παρακολουθεί τη δικτυακή κυκλοφορία και εντοπίζει ύποπτες δραστηριότητες ή πιθανές επιθέσεις.
IncidentΣυμβάνΟποιοδήποτε γεγονός που επηρεάζει την ασφάλεια ενός συστήματος ή δικτύου, το οποίο μπορεί να είναι είτε επίθεση είτε ένα άλλο πρόβλημα ασφαλείας.
Incident ResponseΑντιμετώπιση ΣυμβάντωνΗ διαδικασία αντίδρασης και διαχείρισης μιας κυβερνοεπίθεσης ή άλλου ασφαλιστικού περιστατικού για την ελαχιστοποίηση της ζημιάς και την αποκατάσταση των συστημάτων.
Ethical HackingΗθική ΧάκινγκΗ πρακτική της εκτέλεσης επιθέσεων σε ένα σύστημα με σκοπό την ανίχνευση ευπαθειών πριν από τους κακόβουλους επιτιθέμενους.
PhishingΨαρέμαΜια τεχνική επίθεσης όπου οι επιτιθέμενοι προσπαθούν να εξαπατήσουν τους χρήστες για να αποκαλύψουν ευαίσθητες πληροφορίες, συνήθως μέσω ψεύτικων email ή ιστοσελίδων που μοιάζουν με πραγματικές.
RansomwareΛογισμικό ΑπαγωγήςΈνα είδος κακόβουλου λογισμικού που κρυπτογραφεί τα δεδομένα του χρήστη και απαιτεί λύτρα για την αποκρυπτογράφησή τους.
VulnerabilityΕυπάθειαΜια αδυναμία σε ένα σύστημα ή εφαρμογή που μπορεί να χρησιμοποιηθεί από επιτιθέμενους για να αποκτήσουν μη εξουσιοδοτημένη πρόσβαση ή να προκαλέσουν βλάβη.
Two-Factor AuthenticationΔιπλή ΑυθεντικοποίησηΜέθοδος ασφάλειας που απαιτεί δύο μορφές επαλήθευσης της ταυτότητας του χρήστη, συνήθως έναν κωδικό και έναν επιβεβαιωτικό κωδικό ή βιομετρικό στοιχείο.
Zero-Day ExploitΕκμετάλλευση Μηδενικής ΗμέραςΕπίθεση που εκμεταλλεύεται μια ευπάθεια που δεν έχει αποκαλυφθεί ή διορθωθεί ακόμα από τους δημιουργούς του λογισμικού.
AdwareΛογισμικό ΔιαφημίσεωνΛογισμικό που εμφανίζει ή προβάλλει διαφημίσεις στον υπολογιστή του χρήστη, συνήθως χωρίς την πλήρη γνώση ή συγκατάθεσή του.
BotnetΔίκτυο ΡομπότΈνα δίκτυο από υπολογιστές που έχουν μολυνθεί με κακόβουλο λογισμικό και ελέγχονται εξ αποστάσεως από έναν επιτιθέμενο.
CryptojackingΚρυπτονομισματική ΥποκλοπήΜια επίθεση κατά την οποία οι επιτιθέμενοι χρησιμοποιούν κρυφά τους πόρους ενός υπολογιστή για την εξόρυξη κρυπτονομισμάτων χωρίς τη γνώση του χρήστη.
Remote Access TrojanTrojan Απομακρυσμένης ΠρόσβασηςΜια κατηγορία trojan που επιτρέπει σε επιτιθέμενους να αποκτούν απομακρυσμένη πρόσβαση σε έναν υπολογιστή και να ελέγχουν τις λειτουργίες του.
Social EngineeringΚοινωνική ΜηχανικήΜια τεχνική επίθεσης που βασίζεται στην εκμετάλλευση της ανθρώπινης ψυχολογίας για να πείσει τους χρήστες να αποκαλύψουν ευαίσθητες πληροφορίες.
SQL InjectionΕισβολή SQLΜια τεχνική επίθεσης που εκμεταλλεύεται ευπάθειες σε βάσεις δεδομένων, επιτρέποντας την εκτέλεση κακόβουλων SQL εντολών μέσω εισόδων χρήστη.
Threat IntelligenceΠληροφορίες για ΑπειλέςΔεδομένα και αναλύσεις που συλλέγονται για να εντοπιστούν και να κατανοηθούν πιθανές απειλές στον κυβερνοχώρο, με σκοπό την πρόληψη και την ανίχνευση επιθέσεων.
VishingΦωνητικό PhishingΜια τεχνική επίθεσης μέσω τηλεφώνου όπου οι επιτιθέμενοι προσπαθούν να αποκτήσουν προσωπικές πληροφορίες από τα θύματα μέσω απάτης.
Advanced Persistent ThreatΣύνθετη Επίμονη ΑπειλήΜια επίθεση που στοχεύει σε ένα συγκεκριμένο στόχο με στόχο την παραμονή του επιτιθέμενου στο δίκτυο για παρατεταμένο χρονικό διάστημα.
Brute Force AttackΕπίθεση Μέσω Υπολογισμού Κύριου ΚωδικούΜια μέθοδος επίθεσης που δοκιμάζει κάθε πιθανό συνδυασμό κωδικών πρόσβασης μέχρι να βρει τον σωστό.
BackdoorΚρυφή ΠόρταΈνας μη εξουσιοδοτημένος τρόπος πρόσβασης σε ένα σύστημα, που επιτρέπει στους επιτιθέμενους να παρακάμψουν τα μέτρα ασφαλείας.
Cross-Site ScriptingΔιασταυρούμενη Επίθεση ScriptingΜια επίθεση που επιτρέπει σε επιτιθέμενους να εισάγουν κακόβουλο κώδικα σε ιστοσελίδες που επισκέπτονται άλλοι χρήστες.
Denial of ServiceΆρνηση ΥπηρεσίαςΜια επίθεση που στοχεύει στο να καθιστά ένα σύστημα ή υπηρεσία μη προσβάσιμη με την υπερφόρτωσή του με αίτημα.
Multi-Factor AuthenticationΠολυπαραγοντική ΑυθεντικοποίησηΜέθοδος ασφαλείας που απαιτεί δύο ή περισσότερους παράγοντες για την επαλήθευση της ταυτότητας του χρήστη.
Rootkit-Ένα σύνολο εργαλείων που επιτρέπει σε έναν επιτιθέμενο να αποκρύψει τη δραστηριότητά του και να διατηρήσει την πρόσβαση σε ένα σύστημα.
TokenΚωδικός ΑυθεντικοποίησηςΈνας μηχανισμός που παρέχει ασφαλή πρόσβαση, συνήθως σε συνδυασμό με κωδικούς πρόσβασης ή άλλες μεθόδους αυθεντικοποίησης.
SpywareΛογισμικό ΚατασκοπίαςΛογισμικό που συλλέγει πληροφορίες από έναν υπολογιστή χωρίς την συγκατάθεση του χρήστη.
KeyloggerΚαταγραφέας ΠληκτρολογήσεωνΈνα εργαλείο που καταγράφει τις πληκτρολογήσεις ενός χρήστη με σκοπό την κλοπή ευαίσθητων πληροφοριών, όπως κωδικοί πρόσβασης.
EncryptionΚρυπτογράφησηΗ διαδικασία μετατροπής δεδομένων σε κωδικοποιημένη μορφή ώστε να προστατευτούν από μη εξουσιοδοτημένη πρόσβαση.
ExploitΕκμετάλλευσηΈνα εργαλείο ή τεχνική που χρησιμοποιείται για να επωφεληθεί από μια ευπάθεια σε ένα σύστημα ή εφαρμογή.
Privilege EscalationΑναβάθμιση ΔικαιωμάτωνΗ τεχνική της απόκτησης υψηλότερων δικαιωμάτων πρόσβασης από αυτά που αρχικά επιτρέπονται, συχνά από κακόβουλους επιτιθέμενους.
Root AccessΠρόσβαση ΡίζαςΗ υψηλότερη μορφή πρόσβασης σε ένα σύστημα, που επιτρέπει πλήρη έλεγχο και διαχείριση όλων των πόρων και ρυθμίσεων.
Public Key InfrastructureΥποδομή Δημόσιου ΚλειδιούΈνα σύνολο πολιτικών, διαδικασιών και υλικού που χρησιμοποιείται για τη διαχείριση ψηφιακών πιστοποιητικών και δημόσιων κλειδιών.
Public Key CryptographyΚρυπτογραφία Δημόσιου ΚλειδιούΜέθοδος κρυπτογράφησης που χρησιμοποιεί δύο κλειδιά, ένα δημόσιο και ένα ιδιωτικό, για την ασφαλή επικοινωνία και την επαλήθευση ταυτότητας.
Data EncryptionΚρυπτογράφηση ΔεδομένωνΗ διαδικασία μετατροπής δεδομένων σε κωδικοποιημένη μορφή για να προστατευτούν από μη εξουσιοδοτημένη πρόσβαση.
Token AuthenticationΑυθεντικοποίηση με ΚωδικόΗ χρήση ενός φυσικού ή ψηφιακού κωδικού για την επαλήθευση της ταυτότητας ενός χρήστη ή συσκευής.
Traffic AnalysisΑνάλυση ΚυκλοφορίαςΗ διαδικασία της παρακολούθησης και ανάλυσης της κυκλοφορίας δικτύου για την ανίχνευση ανωμαλιών ή επιθέσεων.
Worm-Ένα είδος κακόβουλου λογισμικού που αναπαράγεται αυτόματα και διαδίδεται μέσω δικτύων χωρίς την παρέμβαση του χρήστη.
URL SpoofingΨεύτικη Διεύθυνση URLΗ τεχνική της παραποίησης μιας διεύθυνσης URL για να φαίνεται ως νόμιμη και να εξαπατήσει τους χρήστες να αποκαλύψουν προσωπικές πληροφορίες.
TokenizationΜετατροπή σε ΚωδικόΗ διαδικασία αντικατάστασης ευαίσθητων δεδομένων με μη ευαίσθητους "κωδικούς" που δεν αποκαλύπτουν την πραγματική πληροφορία.
DNS SpoofingΨεύτικο DNSΗ τεχνική της παραποίησης των εγγραφών DNS για την ανακατεύθυνση των χρηστών σε κακόβουλες ιστοσελίδες.
Digital CertificateΨηφιακό ΠιστοποιητικόΈνα έγγραφο που επιβεβαιώνει την ταυτότητα ενός ατόμου ή οργανισμού μέσω κρυπτογραφικών κλειδιών.
Web Application FirewallΤείχος Προστασίας Εφαρμογών ΙστούΈνα σύστημα που προστατεύει τις εφαρμογές ιστού από επιθέσεις, όπως SQL Injection και Cross-Site Scripting (XSS).
Virtual Private NetworkΕικονικό Ιδιωτικό ΔίκτυοΜια τεχνολογία που δημιουργεί μια ασφαλή και κρυπτογραφημένη σύνδεση μέσω του διαδικτύου, επιτρέποντας την απομακρυσμένη πρόσβαση σε δίκτυα.
Wireless Encryption ProtocolΠρωτόκολλο Κρυπτογράφησης Ασύρματου ΔικτύουΈνα πρωτόκολλο κρυπτογράφησης που χρησιμοποιείται για την προστασία των ασύρματων δικτύων, το οποίο έχει αντικατασταθεί από πιο ασφαλή πρωτόκολλα.
AuthenticationΑυθεντικοποίησηΗ διαδικασία επαλήθευσης της ταυτότητας ενός χρήστη ή συστήματος.
AuthorizationΕξουσιοδότησηΗ διαδικασία καθορισμού των δικαιωμάτων και των επιπέδων πρόσβασης ενός χρήστη σε πόρους ή συστήματα.
Active Directory-Ένα σύστημα καταλόγου της Microsoft που χρησιμοποιείται για τη διαχείριση δικτύων, χρηστών, υπολογιστών και άλλων πόρων σε περιβάλλοντα Windows. Παρέχει κεντρική διαχείριση και ασφαλή αυθεντικοποίηση σε δίκτυα υπολογιστών.
Distributed Denial of Service (DDoS)Κατανεμημένη Επίθεση Άρνησης Υπηρεσίας (DDoS)Μια επίθεση που χρησιμοποιεί πολλούς υπολογιστές για να υπερφορτώσει μια υπηρεσία ή σύστημα, συνήθως μέσω botnets.
Session Hijacking-Μια επίθεση όπου ο επιτιθέμενος αναλαμβάνει τον έλεγχο μιας ενεργής συνεδρίας χρήστη για να αποκτήσει πρόσβαση σε ευαίσθητα δεδομένα.
Web ShellΔιαδικτυακή ΘήκηΈνα κακόβουλο σενάριο που επιτρέπει σε επιτιθέμενους να εκτελούν εντολές σε έναν διακομιστή μέσω μιας διαδικτυακής διεπαφής.
Security Information and Event Management (SIEM)Διαχείριση Πληροφοριών και Συμβάντων Ασφάλειας (SIEM)Συνολικά εργαλεία και τεχνικές για την συλλογή, ανάλυση και αναφορά συμβάντων ασφαλείας σε ένα σύστημα ή δίκτυο.
Security Operations CenterΚέντρο Επιχειρήσεων ΑσφαλείαςΜια μονάδα που παρακολουθεί, αναλύει και αντιδρά σε συμβάντα ασφαλείας σε πραγματικό χρόνο για την προστασία ενός οργανισμού.
