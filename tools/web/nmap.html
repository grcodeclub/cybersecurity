Nmap

Scan a Range of IP Address

nmap 192.168.1.1-24

Scan Port

nmap -p- 192.168.1.5

Για να σαρώσετε τη θύρα TCP 80

nmap -p T:80 192.168.1.5

Για να σαρώσετε τη θύρα UDP 53

nmap -p U:53 192.168.1.5

Ping Scan Using Nmap

nmap -sP 192.168.1.1/20

Σάρωση πιο δημοφιλών ports

nmap -sS 192.168.1.5

Εμφάνιση ανοιχτών ports

nmap –open 192.168.1.5

Ανίχνευση λειτουργικού συστήματος

nmap -O 192.168.1.5

Ανίχνευση έκδοσης υπηρεσίας

nmap -sV 192.168.1.5

Εκτελέστε σάρωση RPC

nmap -sR 192.168.1.5

Εκτελέστε μια επιθετική σάρωση

nmap -A 192.168.1.5

Force Reverse DNS Resolution

nmap -R 192.168.1.5

Disable Reverse DNS Resolution

nmap -n 192.168.1.5

Για σάρωση ολόκληρου του δικτύου

nmap 192.168.1.*
